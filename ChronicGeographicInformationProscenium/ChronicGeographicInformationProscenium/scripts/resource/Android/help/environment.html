系统环境搭建

工具要求系统搭建了Java、Android和Ant环境，需要配置环境变量.

1.搭建Java环境

首先，下载Java开发工具 JDK， 选择与个人系统相符合的版本，下载完成后双击安装。

新建环境变量JAVA_HMOE,并指向你的jdk安装路径，如图：

然后，设置"%JAVA_HMOE%\bin"到系统环境变量的Path下，如图：

至此，Java环境搭建完毕。

2.搭建Android环境

首先，下载Android开发工具 Android SDK For Windows， 下载完成后双击安装。

然后，找到android-sdk-windows目录复制d:\android-sdk-windows\platform-tools;d:\android-sdk-windows\tools;到系统环境变量的Path下，如图：

至此，Android环境搭建完毕。

3.搭建Ant环境

首先，下载Apache Ant辅助打包工具 Apache Ant， 下载完成，解压即可。

然后，找到Apache Ant目录下的bin文件夹"D:\apache-ant-1.8.4\bin",复制该目录配置到系统环境变量的Path下，如图：

至此，Ant环境搭建完毕，需要注意的是Ant环境依赖于Java环境，所以在此之前一定要把Java环境搭建好。
